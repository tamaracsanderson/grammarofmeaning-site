Note the trade-off: the old set says what each section is (a book = library, a quill = writing, a path = a walk, an instrument = a method). The new set is prettier + more unified but leans on the text label to carry meaning (a leaf doesn’t say “library”). You can also mix — e.g. keep the book + quill where meaning matters, use leaves for Garden.
